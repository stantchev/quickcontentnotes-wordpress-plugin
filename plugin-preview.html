📝 Quick Content Notes

Admin-only note-taking meta box for WordPress posts and pages

WordPress Plugin v1.0.0 Admin Only

✨ Key Features

🔒 Admin-Only Visibility

Notes are completely private and only visible to administrators. Never shown to site visitors.

🎨 Color-Coded Priorities

5 priority levels with visual color coding: Default, Important, Idea, Review, and Info.

✅ Status Tracking

Mark notes as Active or Completed to track your progress and workflow.

📝 Markdown Support

Use basic Markdown formatting: bold, italic, links, and lists for better organization.

👁️ Posts List Preview

See note previews directly in your posts/pages list for quick reference.

⚡ Lightweight

Minimal performance impact with clean, efficient code that doesn't slow down your site.

📋 Meta Box in Post Editor

This meta box appears in the sidebar when editing posts or pages (admin users only):

🎨 Priority Color Examples

⚪ Default (Blue) - General notes and reminders for everyday use
🔴 Important (Red) - Urgent items requiring immediate attention
🟡 Idea (Yellow) - Creative ideas or suggestions for future content
🟢 Review (Green) - Items ready for review, approval, or final check
🔵 Info (Light Blue) - Informational notes, references, or context

Simply upload the plugin file to your WordPress installation and activate it. The admin notes meta box will appear immediately in your post and page editors.
